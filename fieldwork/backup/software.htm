Recommendations

I'm recommending the following software tools on the basis of what I've tried out. I welcome suggestions for new software tools - please send them to me using the 'feedback' link. Note that I'm not making this a comprehensive list, and there are programs I know about that I'm not listing here at this stage because they're still in beta release, or I just don't think they do the job well enough.

  • Lexicon, interlinearisation, etc: Toolbox
  • Parsers
    • Alchemist and XX
    • ITE
  • Lexicography
    • TshwaneLex (specifically for dictionary creation)
    • Lexique
  • Phonetics
    • Praat
    • Akustyk
  • Corpus software (in addition to Toolbox)
    • Monoconc
  • Sound editing
    • CoolEdit
    • Audiamus (XX)
    • Audacity
  • Transcribing:
    • Elan
    • Transcriber
    • Childes
    • Transana
  • Keyboard mapping software
    • Keyman
  • Fonts
    • IPA (Gentium, Charis, Doulos)
    • New to Literacy fonts
    • Localised Unicde fonts
    • Aboriginal Sans/Aboriginal Roman
